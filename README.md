# Nagrotech Digital Services | Scroll-Driven Web Application

A modern, high-performance web application built with **React 19**, **TypeScript**, **Tailwind CSS v4**, and a smooth **scroll-driven background video experience**.

---

## Features

- **Full Nagrotech Web Application**: All components including Hero, Services, Why Us, Compare Us, Process, Portfolio, Pricing, Testimonials, FAQ, About Founder, and Contact.
- **Scroll-Driven Video Background**: Smooth, high-definition background video scrubbing synchronized with scroll depth.
- **7 Live Client Projects**: Real visual screenshot glimpses and live external links (`https://www.nagro.in/`, `https://coaching-iit.vercel.app/`, `https://ca-dhanbad.vercel.app/`, `https://emical.online/`, `https://darkslateblue-toad-871981.hostingersite.com/`, etc.).
- **Interactive Price Estimator Modal**: Custom project quote calculator sending formatted specifications straight to WhatsApp.
- **2026 Industry Benchmark Pricing**: Transparent market-rate pricing packages with annual maintenance contract (AMC) toggle options.
- **Direct WhatsApp & Phone Integration**: WhatsApp number `+91 7979783404` wired into all CTA buttons and floating chat widgets.
- **Real Founder Profile**: Authentic photo and bio of Founder & Lead Tech Architect **Raju Naha**.

---

## Tech Stack

- **Frontend Core**: React 19, TypeScript
- **Styling**: Tailwind CSS v4, Glassmorphic UI utilities
- **Build Tool & Server**: Vite 5
- **Icons & Animations**: Lucide-React, Motion (Framer Motion)

---

## Getting Started

### 1. Install Dependencies

```bash
npm install
```

### 2. Run Locally

```bash
npm run dev
```

Open [http://localhost:5173/](http://localhost:5173/) in your browser.

### 3. Build for Production

```bash
npm run build
```
